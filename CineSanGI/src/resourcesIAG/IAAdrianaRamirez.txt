3. If:
“¿Si pongo un if con varias condiciones separadas por ||, se comprueba en el caso en el que uno sea false y en caso de que varios sean false?”
Repuesta:
Cuando utilizas un operador lógico || (OR) en un if, el comportamiento es el siguiente:
1. Cortocircuito del OR (||):
o En cuanto se encuentra una condición verdadera en la secuencia, el resto de las condiciones no se evalúan.
o Si todas las condiciones son falsas, entonces el resultado del if es false.
2. Evaluación paso a paso:
o Se comprueban las condiciones de izquierda a derecha.
o Si una condición es true, se detiene la evaluación y se ejecuta el bloque asociado al if.

5. Obtener fecha en JAVA:
“cómo puedo conseguir la fecha en ese momento en java?”
Respuesta:
En Java, puedes obtener la fecha actual usando la clase LocalDate, LocalDateTime o Date dependiendo de tus necesidades y la versión de Java que estés utilizando. Aquí te dejo algunos ejemplos:
Usando LocalDate y LocalDateTime (Java 8 y versiones posteriores):
Estas clases son parte de la API de tiempo moderno introducida en Java 8.
Obtener solo la fecha actual:
import java.time.LocalDate;
public class Main {
    public static void main(String[] args) {
        LocalDate fechaActual = LocalDate.now();
        System.out.println("Fecha actual: " + fechaActual);
    }
}
